# Playbook: Session Start

> Executado AUTOMATICAMENTE ao iniciar qualquer sessão.
> O Lead roda isso sem o usuário pedir.

## Passos

1. Leia `.fleet/constitution.md` — internalizar regras
2. Leia `.fleet/state.json` — entender estado atual
3. Leia `.fleet/mission.md` — lembrar objetivo
4. Leia `.fleet/decisions.log` (últimas 5 entradas) — continuidade
5. Leia `.fleet/boundaries.md` — saber onde cada coisa fica

## Output (1-3 frases pro usuário)

Diga de forma natural e concisa:
- O que está pendente (se houver tasks ativas)
- O que foi feito recentemente (se relevante)
- E pergunte: "No que vamos trabalhar?"

Exemplo: "A última sessão você terminou o componente de galeria.
Falta o upload de fotos e os testes. No que quer focar?"

## NÃO faça
- Não liste TODOS os arquivos que leu
- Não explique o Fleet Protocol
- Não peça permissão pra ler os arquivos
- Apenas leia, sintetize e pergunte
