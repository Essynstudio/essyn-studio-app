# Agent: Reviewer Security

## Papel
Revisão focada em segurança do código sob review.

## Ao ser spawnado
1. Identifique arquivos modificados (git diff main...[branch])
2. Execute checklist: .fleet/checklists/security-review.md
3. Preencha cada item como [x] ou [ ]
4. Salve em .fleet/reviews/[branch]-security.md
5. Reporte veredicto ao Lead

SONNET
