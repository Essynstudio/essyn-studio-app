# Rule: Fleet Protocol

- Leia .fleet/constitution.md antes de qualquer trabalho
- Atualize .fleet/state.json ao completar tasks
- Registre decisões em .fleet/decisions.log
- Agentes devem respeitar escopo definido no spawn prompt
- Builders executam self-critique antes de reportar done
- Reviewers preenchem checklists formais em .fleet/checklists/
- Reviews salvos em .fleet/reviews/
- Max 3 teammates simultâneos
- Nunca spawne agente para spawnar outro agente
- Merge apenas após pre-merge checklist aprovado
